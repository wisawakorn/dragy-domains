# คู่มือการติดตั้งระบบเครื่องมือจัดการโดเมน .dragy 🚀
ยินดีต้อนรับสู่โปรเจกต์ระบบจดทะเบียนและจัดการโดเมน **.dragy + Google Login + MongoDB + AI Admin** ระบบมีความพร้อมสำหรับนำไปรันใช้งานจริง โดยพัฒนาด้วยสถาปัตยกรรมแบบ Full-Stack (React 19 + Express + MongoDB) ซึ่งมาพร้อมกับระบบฐานข้อมูล 2 โหมดในรูปแบบอัตโนมัติ

---

## 🛠️ จุดเด่นของระบบ
1. **Google Login + Account Security**: ผู้ใช้สามารถยืนยันตัวตนด้วย Google Account ผ่านหน้าต่าง Popup และซิงก์ข้อมูลเครดิต $DRAGY โดเมนที่ถือครอง และรายการประมูลแบบเรียลไทม์
2. **MongoDB Data Persistence**: บันทึกข้อมูลแบบไร้รอยต่อ โดยระบบมีระบบสำรองอัตโนมัติ (Fallback) หากคุณไม่มี MongoDB ดั้งเดิมจำลอง ระบบจะพึ่งพาฐานข้อมูลเชิงลึกแบบ **Local File System JSON Emulator** ช่วยให้รันได้ทันทีโดยไม่ต้องตั้งค่าล่วงหน้า!
3. **MongoDB Compass Admin/Emulator**: มีแดชบอร์ดแอดมินสำหรับการตรวจสอบเอกสารทั้งหมด สถิติการไหลของข้อมูลใน Database คอนโซลรันคำสั่ง Execute คำสั่ง NoSQL และพูลประวัติเวิร์กโหลดคิวรีแบบเรียลไทม์
4. **AI Co-pilot System & Suggester**: ผนวกโมเดล Gemini ในการประเมินราคาโดเมน แนะนำโดเมนอัจฉริยะตามหมวดหมู่ และรับคำสั่งพิมพ์ธรรมดาเพื่อดำเนินการจดรายละเอียด DNS และทำธุรกรรมออโต้

---

## 📋 ความต้องการของระบบ (Prerequisites)
- **Node.js**: เวอร์ชัน 18 ขึ้นไป
- **npm** หรือ **yarn** หรือ **pnpm**
- **MongoDB Atlas** (หากต้องการจัดเก็บบนคลาวด์ภายนอก)

---

## ⚙️ ขั้นตอนการติดตั้งและการรันระบบ

### 1. โคลนและดาวน์โหลดโปรเจกต์
หลังจากเปิดโปรเจกต์ในระบบของคุณ ให้ตรวจสอบความถูกต้องของโครงสร้างไฟล์หลัก:
```bash
/server.ts         # ไฟล์เซิร์ฟเวอร์ Express สำหรับระบบหลังบ้าน
/src/App.tsx       # ส่วนติดต่อผู้ใช้หลักฝั่งหน้าบ้าน React SPA
/tsconfig.json     # การตั้งค่าทรานส์ไพล์เลอร์ TypeScript
/package.json      # ความต้องการของไลบรารีทั้งหมด
```

### 2. ติดตั้ง Dependencies และแพ็กเกจที่จำเป็น
เปิด Terminal ในโฟลเดอร์โปรเจกต์แล้วรันคำสั่งเพื่อสแกนและดาวน์โหลดโมดูล:
```bash
npm install
```

### 3. ตั้งค่าสภาพแวดล้อม (Environment Variables)
แก้ไขไฟล์ `.env` ที่บริเวณ Root Directory (โคลนตัวอย่างจาก `.env.example`):
```env
# คีย์เชื่อมต่อ Gemini AI สำหรับการสแกนและประเมินโดเมน
GEMINI_API_KEY="ใส่คีย์_GEMINI_API_KEY_ของคุณที่นี่"

# เชื่อมต่อฐานข้อมูล MongoDB Atlas (หากเว้นว่างไว้ ระบบจะรันผ่าน Local DB Emulator ให้โดยหน้าบ้านไม่พัง!)
MONGODB_URI="mongodb+srv://ชื่อผู้ใช้:รหัสผ่าน@cluster.mongodb.net/dragy_db?retryWrites=true&w=majority"
```

### 4. รันแอปพลิเคชันในเครื่องคอมพิวเตอร์ของคุณ

**โหมดพัฒนา (Development Mode):**
ทำงานร่วมกับ Hot Reload ทำให้แก้ไขโค้ดและเห็นผลลัพธ์ทันทีผ่านพอร์ต 3000:
```bash
npm run dev
```

**โหมดจำลองโปรดักชัน (Production Mode Build):**
ทำการ Bundle หน้าบ้านให้ไฟล์มีขนาดเล็กลง และคอมไพล์เซิร์ฟเวอร์ให้อยู่ในรูปแบบประสิทธิภาพสูงสุด:
```bash
npm run build
npm start
```
เปิดบราวเซอร์ไปที่: **`http://localhost:3000`** เพื่อเริ่มใช้งาน!

## ❓ คำถามที่พบบ่อย (FAQ)

### ถาม: "ถ้าติดตั้งเสร็จแล้วใช้ Dragy โดเมนได้เลยใช่ไหมครับ?"
**ตอบ:** **ใช่ครับ! ใช้งานได้ทันทีครับ!** 
เมื่อรันระบบเสร็จสิ้น คุณสามารถกดล็อกอินผ่าน Google Login และจดทะเบียนโดมเนใหม่ด้วยนามสกุล `.dragy` โดยใช้เหรียญ $DRAGY ที่ได้รับฟรีจากระบบ Faucet ได้ฟรีทันที ข้อมูลโดเมน บันทึก DNS การเสนอประมูลตลาดกลาง และข้อมูลโปรไฟล์ต่างๆ จะเข้าสู่ฐานข้อมูล MongoDB หรือ Local System จำลองของคุณอย่างเสถียรและทนทาน ไม่หายไปแม้จะปิดหน้าตู้หรือกดรีเฟรชหน้าเว็บก็ตามครับ!
